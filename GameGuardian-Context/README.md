# GameGuardian Context

Sistema de vinculación y supervisión contextual adulto-niño para videojuegos.

## Problema
Plataformas como Roblox, Fortnite, Minecraft tienen:
- Verificación de edad (adulto/niño)
- Pero NO controlan QUÉ adulto interactúa con QUÉ niño

## Solución
Sistema que:
1. **Vincula** adultos específicos con niños específicos
2. **Requiere** autorización mutua para cada sesión
3. **Supervisa** contextos y tiempos
4. **Alertas** a padres en tiempo real
